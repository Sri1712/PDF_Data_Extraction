At December 31, 2025 and 2024, the difference between the carrying amount andestimated fair value for loans and long-term receivables was not material. If measured at fair value in the financial statements,these financial instruments would be classified as Level 3 in the fair value hierarchy.## Long-Term DebtThe majority of the company’s long-debt portfolio is comprised of publicly traded debt, and its fair value is based on quoted marketprices for the identical liability when traded as an asset in an active market (Level 1). For other long-term debt (including long-termfinance lease liabilities) for which a quoted market price is not available, an expected present value technique that uses ratescurrently available to the company for debt with similar terms and remaining maturities is used to estimate fair value (Level 2).
